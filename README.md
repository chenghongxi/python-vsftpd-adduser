# python-vsftpd-adduser
此脚本为了满足批量添加vsftp用户以及数据迁移

使用方法：

1. tar zxvf PyMySQL-0.8.1.tar.gz && cd PyMySQL-0.8.1
2. python setup.py install
3. tar zxvf xlrd-1.2.0.tar.gz && cd xlrd-1.2.0
4. python setup.py install
#python依赖包安装完毕
